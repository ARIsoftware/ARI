## Quick Start

Install ARI with a single command:

**macOS / Linux:**
```bash
/bin/bash -c "$(curl -fsSL https://ari.software/install)"
```

**Windows** (PowerShell):
```powershell
irm https://ari.software/install-win | iex
```

The installer handles everything — dependencies, database setup, and configuration.
